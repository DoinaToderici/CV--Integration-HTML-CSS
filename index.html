A propos de moi

Moi c'est Doina,

Je suis une développeuse web axée sur la conception. Je crée de belles sites Web à l'aide d'un code soigneusement conçu et d'une conception centrée sur l'utilisateur.
Mon métier n'est pas seulement d'imaginer et de créer de jolis concepts web , mon métier c'est aussi de faire en sorte que le produit en ligne soit le plus efficace et le plus intuitif possible. Le produit doit convertir. Le produit doit se différencier.
